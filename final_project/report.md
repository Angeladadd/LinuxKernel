解决：

```bash
git clone https://github.com/memkind/memkind.git
apt-get install autoconf automake libtool unzip libnuma-dev numactl
cd memkind
./autogen.sh
./configure --prefix=/usr
make
make install
```

```
./heat

```

查看core dump原因

```bash
ulimit -c unlimited #让系统产生core文件
ulimit unlimited
apt install gdb #安装gdb
./heat
gdb heat core
```
